## 基础规则
所有布局元素都拥有以下属性:
1. Margin 外边距
2. Padding 内边距
3. 横向对齐   
    Start,Center,End,Fill  
4. 纵向对齐  
    Start,Center,End,Fill
5. 是否跳出布局流
6. 强制大小  
默认情况下不推荐指定元素的大小，布局系统会根据需求动态调整大小。  
也可以手动指定强制大小。

## 支持的组件
1. StackPanel
2. ContentView
3. Grid
4. FlexPanel
5. Canvas
